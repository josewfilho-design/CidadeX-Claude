# Memory: features/agenda/lista-de-atividades
Updated: now

A aba 'Lista de Atividades' (renomeada de 'Lista de Compras') dentro da Agenda permite criar listas nomeadas por data para organização de itens. Cada item suporta nome, quantidade, unidade, valor estimado, categoria e status de conclusão (checkbox). As categorias são dinâmicas: incluem categorias padrão (Frutas, Verduras, Carnes, etc.) e categorias customizadas criadas pelo usuário via opção "+ Nova categoria..." nos selects, persistidas na tabela `user_custom_options` com `option_type = 'shopping_category'`. Categorias customizadas podem ser renomeadas e excluídas. A interface fornece barras de progresso por lista e cálculos automáticos de valor total. O sistema permite drag-and-drop para reordenação, marcar/desmarcar todos, filtros avançados (data, status, categoria, busca), compartilhamento WhatsApp e exportação PDF (abre no /visualizador em tela cheia com opções de imprimir/salvar). Por regra do sistema, a funcionalidade de importar/exportar JSON foi removida. Limite de 50 listas por usuário.
